# Cake-Assignment

Recommend to use Node.js v16.

```sh
nvm install 16
```

To use Node v16

```sh
nvm use 16
```

Before starting the application

```sh
npm i
```

To start the application in dev mode

```sh
npm start
```

To run the application with `npx`

```sh
npx ts-node index.ts
```

To run unit tests.

```sh
npm test
```
